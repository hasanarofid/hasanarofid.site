# hasanarofid.site

Landing page portofolio untuk Hasan Arofid (Senior Fullstack Engineer) dengan tampilan elegan, dark theme, dan lightbox untuk menampilkan portofolio secara penuh.

## Fitur
- Hero section dengan CTA (email & GitHub).
- Daftar layanan dan badge core stack.
- Grid portofolio dengan gambar lokal (`images/porto*.png`) dan lightbox interaktif.
- Desain responsif bernuansa glassmorphism.

## Struktur
- `index.php` — halaman utama statis.
- `images/` — aset gambar portofolio.

## Menjalankan secara lokal
Tidak ada dependency build. Cukup layankan folder ini menggunakan server statis atau PHP bawaan:
```bash
php -S localhost:8000
```
Lalu buka `http://localhost:8000/index.php`.

## Deployment
Karena statis, dapat ditempatkan di layanan static hosting (Netlify, Vercel, GitHub Pages dengan PHP dihilangkan) atau dilayani lewat server PHP/Apache/Nginx.
